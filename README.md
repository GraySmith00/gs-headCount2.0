# HeadCount 2.0 by Gray Smith

HeadCount 2.0 is a dynamic application that will show you all of the shool district data in Colorado for many different categories.

## To Get Started

Clone the repository using:

```
git clone git@github.com:GraySmith00/headcount2.0.git
```

In the terminal run:

```
npm install
```

To start the application, run:

```
npm start
```

To start the testing suite, run:

```
npm test
```

To run eslint, run:

```
npm run eslint
```
